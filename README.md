# **Cetvel**
> ARKit ile yapılmış, uzunluk ve alan ölçebilen bir cetvel uygulaması

## Requirements

- Swift 4.0
- iOS 11.0
- Xcode 9.0

## Pods

- OpenCV
- LiquidFloatingActionButton
- SnapKit

## Preview

1. Mesure Length

## Install

- Manually compile
1. `git clone https://github.com/kurtulusahmet/Cetvel`
2. `pod install`
3. Compile project with Xcode
